# QOYOD Design Template Library

Each template is a fixed layout pattern. The agent picks ONE template per design based on the brief, then fills in the variables. This guarantees consistent quality and prevents the agent from inventing weird layouts.

---

## Template 1: `hero_person_right`
**Use when:** Showcasing the brand with a human face — testimonial, trust-building, CEO-style messaging
**Aspect ratios:** 1:1, 9:16, 4:5
**Reference:** Image 1 of your reference batch

```
Layout zones:
- Top 40%: Headline + subheadline, right-aligned (RTL)
- Bottom 60%: Saudi businessman in white thobe + red shumagh, 
  centered or right-of-center, looking at camera
- Bottom corners: qoyod.com (left) + QOYOD logo (right)
- Optional: ZATCA logo bottom-center

Background: Deep navy (#0A1F44) gradient, subtle cyan glow 
behind subject's head

Mood: Trustworthy, confident, premium
```

## Template 2: `device_showcase_split`
**Use when:** Highlighting a product feature shown on screen — invoicing, dashboard, mobile app
**Aspect ratios:** 1:1, 16:9
**Reference:** Image 4, 11 of your reference batch

```
Layout zones:
- Right 50%: Realistic device mockup (laptop / phone / tablet) 
  showing a QOYOD UI screenshot, floating with subtle drop shadow
- Left 50%: Headline (large, top), subheadline (below), 
  CTA button (bottom)
- Bottom corners: qoyod.com (left) + QOYOD logo (right)

Background: Solid deep navy (#0A1F44) with concentric circle 
pattern emanating from device, OR clean white-to-blue gradient

Mood: Product-focused, modern, technical
```

## Template 4: `illustration_split`
**Use when:** Explaining a concept playfully — onboarding, "how it works", educational

```

  abstract device illustration

the illustration zone from text zone
```

## How to pick a template

| If the brief is about... | Use template |
|---|---|
| Building trust / human connection | 1 (hero_person_right) |
| Showing the product UI | 2 (device_showcase_split) |
| Multi-platform pitch | 3 (dual_device_hero) |
| Educating / explaining a feature | 4 (illustration_split) |
| Emotional pain point / testimonial | 5 (full_photo_overlay) |
| Discount or special offer | 6 (pricing_offer) |
| ZATCA / government news | 7 (government_partnership) |
| Listing service benefits | 8 (service_features_card) |
| Urgent CTA / announcement | 9 (bold_typographic) |
| Brand statement / mission | 10 (text_dominant_modern) |
